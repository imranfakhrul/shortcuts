# Chrome

## Open a new tab

`CTRL + T`

## Open last closed tab

`CTRL + SHIFT + T`

## Open a new window

`CTRL + N`

## Open a incognito window

`CTRL + SHIFT + N`

## Open download page

`CTRL + J`

## Open history page

`CTRL + H`

## Close a tab

`CTRL + W`

## View a page source

`CTRL + U`

## Reload a page

`CTRL + R`

## Hard reload a page

`CTRL + SHIFT + R`

## Focus address bar

`CTRL + L`

## Rearrange/Move tabs left

`CTRL + SHIFT + PGUP`

## Rearrange/Move tabs right

`CTRL + SHIFT + PGDN`

## Swich tab to the right

`CTRL + TAB`

## Swich tab to the left

`CTRL + SHIFT + TAB`

## Switch to a (n)th tab (works for first 8 tabs)

`CTRL + [TAB NUMBER]`

ex: `CTRL + 6` (switch to 6th tab)

## Switch to a last (most right) tab

`CTRL + 9`

## Print a document/page

`CTRL + P`

## Bookmark a page

`CTRL + D`
